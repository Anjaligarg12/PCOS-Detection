# 🩺 AI-Based PCOS Detection using Deep Learning

## 📌 Overview
This project is an AI-powered PCOS Detection System that predicts the possibility of Polycystic Ovary Syndrome (PCOS) using Machine Learning and Deep Learning models. It also provides diet recommendations based on the prediction.

## ✨ Features
- PCOS Prediction using AI
- Deep Learning Model
- Personalized Diet Recommendation
- User-Friendly Interface
- Fast Prediction

## 🛠 Tech Stack
- Python
- TensorFlow / Keras
- Scikit-learn
- Pandas
- NumPy
- Flask
- HTML/CSS

## 📂 Project Structure

```
PCOS-Detection
│── model/
│── app.py
│── requirements.txt
│── test.py
│── home.jpg
```

## 🚀 Installation

```bash
git clone https://github.com/Anjaligarg12/PCOS-Detection.git
cd PCOS-Detection
pip install -r requirements.txt
python app.py
```
